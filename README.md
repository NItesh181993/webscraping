# webscraping
Webscraping using Node js 

What you need to run this app:

* [Node.Js](https://nodejs.org)

## Instructions

### Setup

* Clone this repo:


* Run this command:

```
npm install
```

### Running the code

### 1) Execute this command in your terminal:

```
node index.js
```
